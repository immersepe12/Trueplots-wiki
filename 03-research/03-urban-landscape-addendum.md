## Urban Sites — Competitive Landscape

### Direct competitors

**No established fractional / joint-buying platform for urban residential plots in Bangalore.** This whitespace is structurally similar to the farmland finding — no Indian player matches buyers, verifies, executes sale, subdivides, and delivers individual khatas per buyer as a single integrated flow. Adjacent services exist unbundled (khata bifurcation by Khata Broker, EzyLegal, Bino.bot at ₹5–15K; legal verification by independent lawyers at ₹10–25K), but no marketplace stitches these into an end-to-end product.

**However, the urban competitive landscape is materially more crowded on verification than farmland:**

- **NoBroker** (6,768 Bangalore plot listings) — the single largest fast-follow threat. Already offers advocate-based legal verification as a paid upsell. Owner-direct positioning aligns naturally with TruePlots' disintermediation narrative. Well-funded (Tiger Global, Accel, General Atlantic). **Could ship a fractional + deeper-verification feature within one quarter** if TruePlots validates demand.
- **99acres** — ~15,367 Bangalore plot listings. Self-declared "verified" badge is marketing-grade. Could upgrade to legal-grade within 12 months, but fractional is philosophically off-brand for InfoEdge's lead-gen DNA.
- **MagicBricks, Housing.com, Square Yards, Homes247** — secondary listings portals. SEO/paid-ads distribution competitors, not product-feature copyists.
- **Aasaan, PropertyGully, PlotStore** (named in earlier scoping) — not material competitors. Aasaan is a small-business listing tool; PropertyGully and PlotStore show no material Bangalore operations.

### Indirect competitors — layout developers

- **Prestige, Brigade, Sobha** — premium plotted developments (Devanahalli, Malur). Entry ₹1.2Cr+. First-sale solo-buyer only. No joint-buying innovation; unwilling to fragment premium brand.
- **NBR Group** and BMRDA-approved layout sellers — dominant mid-market. NBR could bolt fractional onto unsold 60×40 inventory (pre-subdivided as 2×30×40 SKUs) using their existing BBMP khata bifurcation relationships. Low motivation today; **HIGH capability if TruePlots validates demand** — could ship fractional on captive stock within 6–9 months.

### Status quo / incumbents

- Independent brokers: 1–2% commission each side; opaque, relationship-based
- Family / friends informal joint-purchase attempts: visible in Kaanoon and Quora threads but no productised path; failure rate high because partition deeds + khata bifurcation aren't standardized for buyer-facing consumption

### Fractional CRE (not a direct competitor)

SEBI SM REIT framework (2024) products — hBits, Strata, PropShare, Assetmonk — are pooled SPV investments in **commercial** property. Investors receive units/shares, not land titles. Structurally different from TruePlots urban plots with individual titles. Relevance: regulatory context (TruePlots avoids SM REIT scope by design) and operational lessons on exit/liquidity design.

### Key regulatory differences (urban vs farmland)

| | Farmland | Urban sites |
|---|---|---|
| RERA applicability | Excluded | **Applies to plotted developments >500 sqm or >8 units** |
| Ownership restriction | Karnataka 79A/79B (repealed 2020, politically contested) | None — any resident can buy |
| Khata / title system | RTC / Bhoomi | BBMP A-khata / B-khata / BDA / BMRDA / BIAPPA |
| Subdivision process | 11E sketch + phodi + mutation (3–9 months) | Khata bifurcation + BBMP approval + partition deed (3–6 months) |
| Minimum plot size | Fragmentation Act (often 1–2 acres) | BBMP typically 600 sqft |
| Stamp duty (Karnataka, 2026) | Same 5% >₹45L | Same 5% >₹45L |
| Registration charge | 2% (doubled Aug 2025) | 2% (doubled Aug 2025) |

**Critical implication:** If TruePlots positions as **facilitator** (buyer-matching + verification + transaction coordination), RERA scope is lighter — agent registration only. If TruePlots takes an **SPV/promoter** role in subdivisions, **RERA project registration triggers**. This is a **structural question requiring legal counsel opinion before month-4 launch**, not after.

### Fast-follow threat ranking (12-month horizon)

1. **NoBroker** — HIGH: verification deepening + fractional both plausible in 6–12 months
2. **NBR Group / BMRDA layout developers** — MEDIUM: fractional on captive stock in 6–9 months
3. **99acres** — MEDIUM on verification upgrade only

**Defensibility posture:** The moat in urban sites cannot be the marketplace UI or the verification report — both replicable. Moat must be **operational muscle on BBMP khata bifurcation + buyer matching + escrow-based transaction architecture**. A marketplace-only TruePlots gets copied by NoBroker in one quarter.
